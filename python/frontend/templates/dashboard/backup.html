Backup & Restore

Manage your data backups

Data Export

Your data is stored in PostgreSQL with automated Docker volume persistence. Use the commands below to create manual database backups.

# Export full database backup
docker exec voiceflow-postgres pg_dump -U vf_admin voiceflow_prod > backup_$(date +%Y%m%d).sql
# Restore from backup
docker exec -i voiceflow-postgres psql -U vf_admin voiceflow_prod < backup_20250101.sql
